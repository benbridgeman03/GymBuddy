A simple mobile gym tracking app built with Flutter and Dart, using Firebase as the backend.

The app allows users to:
  Log workouts and exercises
  Track sets, reps, and weights
  View and analyze past workout history

This project is currently under development. The goal is to provide an easy and intuitive way to track gym progress for Android.

Tech Stack
  Flutter & Dart
  Firebase Authentication & Firestore
  Provider (state management)
